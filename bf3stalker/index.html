Opt-out

Despite the fact that the data we present is accessible to anyone with a web browser, we have decided to allow an opt-out from our services. Number one reason being that DICE really should implement some privacy features, but also because there has been concern from certain individuals, that the mere existence of our website has caused an increase in "griefing" for these players. While we are skeptical, we believe the right thing to do is allow players to decide themselves whether they wish to be here or not :)

If you wish to be removed (or added) from our site, please contact us at admin@bf3stalker.com.

The following players have requested to be removed thus far:

  • LevelCap
  • InTheWorks
  • JackFrags

Non-profit

This site is hosted in the cloud (Amazon EC2), behind CloudFlare.

Technologies used include the Amazon AMI, Apache, PHP, Google Analytics, New Relic, Bootstrap, phpQuery, jQuery, JSON & EC2 api tools for daily snapshots. To simplify & save memory, we avoid SQL by storing our player data in a .csv file.

A message for EA/DICE

This message is specifically written for EA/DICE, who might find this kind of service to be against their interests. Our site does nothing besides download (crawl) specific players Battlelog pages to investigate what server they are playing on. We have a cron task set up for this, that is executed every 5 minutes. Considering that a single Battlelog profile, on average, has a size of about 15kB, we generate 15 * 160 = 2400 kilobytes per 5 minutes, which is 2400 / 5 = 480 kilobytes per minute. That is equivalent to 480 / 60 = 8 kilobytes per second. Keep in mind that we are requesting all data compressed with gzip, so you can, on average, lower the number by about 60%. We sincerely hope that you will allow us to continue with running this informative service, and encourage people to go play Battlefield with their idols. Thanks!
